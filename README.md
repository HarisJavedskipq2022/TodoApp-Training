## Installation

Install the project with npm

```bash
  npm install 
```

    
## Run Locally

Clone the project

```bash
  git clone https://github.com/HarisJavedskipq2022/TodoApp-Training
```

Go to the project directory

```bash
  cd TodoApp-Training
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```


## Running Tests

To run tests, run the following command

```bash
  npm test
```
